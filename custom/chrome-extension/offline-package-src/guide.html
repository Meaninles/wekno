WeKnora 配置与部署指南
本插件支持连接官方托管的 WeKnora 服务，也支持连接你自己私有化部署的 WeKnora 实例。如果你还没有可用实例，建议先参考 Tencent/WeKnora 完成部署；如果你使用官方托管服务，可直接访问 https://weknora.weixin.qq.com。
1 准备 WeKnora 实例
如果你还没有可用实例，请先参考 GitHub 上的 WeKnora 项目 完成私有化部署；如果你使用官方托管服务，可直接打开 https://weknora.weixin.qq.com。
2 登录你的 WeKnora 实例
打开你自己的 WeKnora 地址并完成注册/登录。后续在插件中填写的服务地址，应为该实例对应的 API 地址，例如 https://your-weknora-domain/api/v1。
3 找到「全部设置」
登录后，在页面左下角找到并点击「全部设置」入口，进入用户设置页面。
全部设置
4 打开「API 信息」
在设置页面中，找到并点击「API 信息」选项卡，进入 API 管理页面。
全部设置 › API 信息
5 复制 API Key
在 API 信息页面中，找到 API Key 字段，点击旁边的复制按钮将其复制到剪贴板。
请妥善保管您的 API Key，不要泄露给他人
6 粘贴到插件中
返回本插件的 WeKnora 登录页面，将你的实例 API 地址粘贴到「服务地址」，将复制的 API Key 粘贴到「API Key」输入框中，点击「保存并登录」即可完成配置。
如果遇到问题，请先确认服务地址是否为你的 WeKnora 实例 API 地址，格式通常为 https://your-weknora-domain/api/v1；如果你使用官方托管服务，可填写 https://weknora.weixin.qq.com/api/v1。
